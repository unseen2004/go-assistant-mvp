# Go Assistant MVP

## Introduction
This is a minimal viable product (MVP) for the Go Assistant application, which aims to assist users in managing tasks efficiently.

## Features
- Task management
- User-friendly interface
- Notifications and reminders

## Installation
1. Clone the repository:
   ```
   git clone https://github.com/unseen2004/go-assistant-mvp.git
   ```
2. Navigate to the project directory:
   ```
   cd go-assistant-mvp
   ```
3. Run the application:
   ```
   go run main.go
   ```

## Usage
- Add new tasks using the "Add Task" button.
- View and manage existing tasks in the tasks list.
